MENTOR

Пройдите путь к пробуждению вместе с близким в роли наставника. Основной задачей ментора является построение лабиринта через собственную призму восприятия курса. Послушник, кем является ваш близкий, обязан заглянуть в каждый тупик вашего лабиринта и под условным руководством найти верный путь, осознавая, что он под присмотром родственной души. В данном видео курсе взята за основу методика «Awakening». Фильм «Mentor» предусматривает просмотр только для наставника. Для полного понятия и ознакомления может потребоваться несколько просмотров.

  • · Видео курс Awakening ( теория + парктика + экзамен )
  • · Видео Курс Mentor ( обучение наставничеству )
